# SDET-Intern-Assessment
Write a script in Python or JavaScript that: 1. Reads both CSV files 2. Compares them based on a primary key 3. Detects: missing rows (rows present in one file but not the other), mismatched values in same fields 4. Outputs: a simple report (console, CSV, or word, excel) including: total mismatches, total missing rows, details of mismatched fields
